# Route C C10 年度评分排名设计

## 目标

为科室负责人/小组负责人新增一个低频、独立的 `年度评分排名` 菜单，用于查看指定年度内每位员工四个季度实际得分的综合排名。

## 交互范围

- 菜单位置：负责人菜单组下，位于 `评分排名` 后
- 页面能力：
  - 年度选择
  - 员工搜索
  - 年度排名列表
  - 右侧员工年度明细

## 排名口径

- 年度总分 = `Q1 + Q2 + Q3 + Q4` 四个季度的实际得分之和
- 缺失季度按 `0` 计
- 不区分 `A+ / A / B+ / B / C`
- 只按年度总分排序，同分时按员工姓名排序

## 权限

- 科室负责人/小组负责人均可查看所有员工年度排名
- 该页面仅查看，不涉及评分写操作

## 返回数据

- 排名列表项：
  - 员工
  - 科室
  - 小组
  - 年度总分
  - Q1~Q4 分数
- 选中员工详情：
  - 员工基础信息
  - 年度总分
  - Q1~Q4 分数卡片

## 验证

- 后端 e2e：年度总分按四季度求和，缺失季度按 0
- 前端 helper 测试：筛选、默认选中、年度总分格式化
- 前端页面测试：菜单可见、页面可渲染、排名列表和季度卡片展示
